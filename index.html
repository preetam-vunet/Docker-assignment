❓ Frequently Asked Questions (FAQ)

1️⃣ What is Markdown?
Markdown is a lightweight markup language that you can use to add formatting elements to plaintext documents.
2️⃣ What is the difference between Markdown and HTML?
Markdown is simpler and easier to read in raw form, while HTML is more powerful and flexible but verbose.
3️⃣ Can I use Markdown for creating websites?
Yes! Markdown can be converted into HTML and used in static site generators like Jekyll, Hugo, or MkDocs.
4️⃣ How to add images in Markdown?
Use ![Alt text](image_url).
Example: ![Logo](https://example.com/logo.png)
5️⃣ Does GitHub support Markdown?
Yes! GitHub supports Markdown and extends it with GitHub Flavored Markdown (GFM), adding tables, task lists, and syntax highlighting.
